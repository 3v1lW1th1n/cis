# IBM Cloud CIS 白名單 IP 位址
{:#cis-whitelisted-ip-addresses}

下列 API 列出與原點進行通訊時 CIS 邊緣伺服器所使用的所有 IP 位址：[https://api.cis.cloud.ibm.com/v1/ips](https://api.cis.cloud.ibm.com/v1/ips)。一週輪詢一次這個 API，就足以取得更新白名單時所需的資訊。
