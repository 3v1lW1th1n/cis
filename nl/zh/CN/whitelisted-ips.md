# IBM Cloud CIS 白名单 IP 地址
{:#cis-whitelisted-ip-addresses}

以下 API 列出 CIS Edge 与源通信时所使用的所有 IP 地址：[https://api.cis.cloud.ibm.com/v1/ips](https://api.cis.cloud.ibm.com/v1/ips)。每周轮询一次此 API 就足以获取更新白名单所需的信息。
