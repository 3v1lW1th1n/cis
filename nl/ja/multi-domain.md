# マルチドメイン・サポート
{:#multi-domain-support}

IBM CIS は、Enterprise Package プランと Enterprise Usage プランで、複数のドメインをサポートしています。 

追加のドメインを CIS インスタンスに追加するには、**「概要」**ページに移動します。**「ドメイン」**セクションの下で、ドメインの右側にあるプラス・アイコンを選択します。これにより、追加のドメインを接続するセクションが示されます。セクション内の手順に従って、新しいドメインの追加を完了します。 

  * ドメインの接続
  * DNS レコードの設定
  * ドメイン管理の代理委任
  
新しいドメインを追加した後に、**「ドメイン」**セクションの下のドロップダウン・メニューを使用して、ドメインを切り替えます。**「ドメイン」**セクションは、**「概要」**ページ、および各ページの上部にあります。ドメインを切り替えることにより、各ドメインの設定を構成できます。
